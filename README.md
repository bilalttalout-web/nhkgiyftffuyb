# Portfolio Projects

This repository contains two interactive web development projects:

## 1. Nike Landing Page (`nike_landing.html`)
A premium, animated e-commerce landing page concept for Nike.
**Features:**
- **Dynamic Theme Switching:** Changes color schemes for Men, Women, and Kids categories.
- **Glassmorphism:** Modern UI with frosted glass effects.
- **Interactive Animations:** Smooth transitions, hover effects, and parallax elements.
- **Responsive Design:** Fully optimized for desktop and mobile.

### How to Run:
Simply open `nike_landing.html` in any modern web browser.

---

## 2. Ouarzazate Bus Tracker (`index.html`)
A functional prototype for a city bus tracking application.
**Features:**
- **Interactive Map:** Uses Leaflet.js to display routes on an OpenStreetMap layer.
- **Route Visualization:** Shows bus paths, stops, and estimated times.
- **Real-time Simulation:** Simulates bus movement along the routes.

### How to Run:
Open `index.html` in your web browser.

## Installation
To download and run these projects locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd your-repo-name
   ```

3. **Open the files:**
   Double-click `nike_landing.html` or `index.html` to view the projects.
